## Giới thiệu

Đây là **nội dung mẫu** viết bằng Markdown. Sau khi chạy `npm run build`, file `index.html` sẽ được tạo cạnh `index.md`.

### Bạn có thể

- Chèn link nội bộ như [trang chủ](/) hoặc [liên hệ](/lien-he/).
- Dùng danh sách:

1. Viết bài trong Decap CMS
2. Lưu (commit lên GitHub)
3. GitHub Actions chạy build và deploy

> Sau khi cấu hình OAuth GitHub cho Decap, mở `/admin/` trên domain để đăng bài.
